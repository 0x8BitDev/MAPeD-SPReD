Graphics Banks Data




Each graphics bank contains: CHR bank, 2x2 tiles (blocks) (256), 4x4 tiles (256), palettes and screens data.




Int32 mpd_num_banks()



Get a number of graphics banks in a project.


Returns:

the number of graphics banks in a project




Int32 mpd_get_active_bank()



Get an active graphics bank index.


Returns:

the index of an active bank




Array[UInt32] mpd_get_tiles( Int32 _bank_ind )



Get 4x4 tiles array of an active bank.


Parameters:

_bank_ind – graphics bank index


Returns:

the 4x4 tiles array or null in case of incorrect bank index


Each tile value (UInt32) consists of blocks indices ordered left to right, up to down from low to high byte ( 4 bytes )




Array[UInt16] mpd_get_blocks( Int32 _bank_ind )



Get blocks ( 2x2 tiles ) array of an active bank.


Parameters:

_bank_ind – graphics bank index


Returns:

the blocks data array or null in case of incorrect bank index


Each block's value (UInt16) has the following bits description: [ 15-8 -> property_id(4) palette ind(2) not used(2) | CHR ind(8) <-- 7-0 ]. Thereby, four UInt16 values form one block ( 2x2 tile ).




Array[Byte] mpd_get_CHR_data( Int32 _bank_ind )



Get raw CHR data. CHR data is stored as image 128x128 (8bit).


Parameters:

_bank_ind – graphics bank index


Returns:

the array of bytes ( 16K ) or null in case of incorrect bank index




Long mpd_export_CHR_data( Int32 _bank_ind, String _filename, Bool _need_padding )



Convert CHR bank to NES format and save it to a file.


Parameters:

_bank_ind – graphics bank index

_filename – exported data filename

_need_padding – save padding data to 1/2/4 KB or not


Returns:

the size of saved file in bytes




Array[Byte] mpd_get_palette( Int32 _bank_ind, Int32 _plt_ind )



Get a 4 colors pallete by input index.


Parameters:

_bank_ind – graphics bank index

_plt_ind – palette index


Returns:

the 4 bytes array of color indices or null in case of incorrect bank index




Int32 mpd_num_screens( Int32 _bank_ind )



Get a number of screens in a graphics bank.


Parameters:

_bank_ind – graphics bank index


Returns:

the number of screens created in a graphics bank or -1 in case of incorrect bank index




Array[Byte] mpd_get_screen_data( Int32 _bank_ind, Int32 _scr_ind )



Get an array of 4x4 tiles indices which form screen data.


Parameters:

_bank_ind – graphics bank index

_scr_ind – screen index


Returns:

the bytes array of 4x4 tiles indices of a particular screen or null in case of incorrect bank index



Layouts Data




Each layout is a matrix of cells. Each cell can contains a screen index and marks ( adjacent screens mask + screen property value 0...15 ).




Int32 mpd_num_layouts()



Get a number of layouts available in a project.


Returns:

the number of layouts




Int32 mpd_layout_width( Int32 _layout_ind )



Get a layout width.


Parameters:

_layout_ind – layout index


Returns:

the input layout width or -1 in case of incorrect layout index




Int32 mpd_layout_height( Int32 _layout_ind )



Get a layout height.


Parameters:

_layout_ind – layout index


Returns:

the input layout height or -1 in case of incorrect layout index




Int32 mpd_layout_start_screen_cell( Int32 _layout_ind )



Get a cell index which contains a start screen index.


Parameters:

_layout_ind – layout index


Returns:

the cell index which contains a start screen index or -1 in case of incorrect layout index


To get a start screen index you can do the following steps:


layout_width = mpd_layout_width( layout_n )

layout_height = mpd_layout_height( layout_n )

start_screen_cell = mpd_layout_start_screen_cell( layout_n )

start_screen_ind = mpd_layout_screen_ind( layout_n, start_screen_cell % layout_width, start_screen_cell / layout_width ) )






SByte mpd_layout_screen_ind( Int32 _layout_ind, Int32 _scr_pos_x, Int32 _scr_pos_y )



Get a screen index by cell coordinates ( zero-based values ).


Parameters:

_layout_ind – layout index

_scr_pos_x – cell index by X axis

_scr_pos_y – cell index by Y axis


Returns:

the screen index in a common screens array of all screens of all graphics banks ( in the editor you can set the 'All Banks' checkbox ( Layout tab → Screens tab ) to see that common screens array ) or -1 in case of incorrect layout index




UInt16 mpd_layout_screen_marks( Int32 _layout_ind, Int32 _scr_pos_x, Int32_scr_pos_y )



Get screen marks assigned to a screen with particular coordinates ( zero-based values ).


Parameters:

_layout_ind – layout index

_scr_pos_x – cell index by X axis

_scr_pos_y – cell index by Y axis


Returns:

the screen marks value: 7-4 - bit mask of user defined adjacent screens ( Down(7)-Right(6)-Up(5)-Left(4) ); 3-0 - screen property

or 0xFFFF in case of incorrect layout index



Entity Instances




Entity instances inherit all data of base entities and they are stored in screen data.




Entity instance class:



public class mpd_py_inst_entity

{

// autogenerated unique id

public Int32 uid;


// unique id of a target entity

public Int32 target_uid;


// instance coordinates

public Int32 x;

public Int32 y;


// base entity id

public Int32 base_ent_uid;


// instance properties

public String properties;

}




Int32 mpd_layout_screen_num_entities( Int32 _layout_ind, Int32 _scr_pos_x, Int32 _scr_pos_y )



Get a number of entities in a screen with particular coordinates.


Parameters:

_layout_ind – layout index

_scr_pos_x – cell index by X axis

_scr_pos_y – cell index by Y axis


Returns:

the number of entities or -1 in case of incorrect layout index




mpd_py_inst_entity mpd_layout_get_inst_entity( Int32 _layout_ind, Int32 _scr_pos_x, Int32 _scr_pos_y, Int32 _ent_ind )



Get an instance entity data.


Parameters:

_layout_ind – layout index

_scr_pos_x – cell index by X axis

_scr_pos_y – cell index by Y axis

_ent_ind – entity index


Returns:

the mpd_py_inst_entity object or null in case of incorrect layout index



Base Entities




Base entity class:



class mpd_py_base_entity

{

// user defined unique id

public Byte uid;


// entity dimensions

public Byte width;

public Byte height;


// origin offset

public SByte pivot_x;

public SByte pivot_y;


// string name

public String name;


// base properties inherit by all instances

public String properties;


// instance property by default (can be overridden by each instance)

public String inst_properties;

}

Parameters:

mpd_py_base_entity mpd_get_base_entity_by_name( String _entity_name )



Get a base entity by name. You can use this function if you know a base entity name exactly.

the mpd_py_base_entity object or null in case of incorrect group name
