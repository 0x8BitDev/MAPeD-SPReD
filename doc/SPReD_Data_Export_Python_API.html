Main Functions




Bool spd_8x16_mode_enabled()



Retrieve the 8x16 mode status.


Returns:

True – enabled; False – disabled




Array[Int32] spd_get_palette( Int32 _plt_ind )



Get a 4 colors palette by index.


Parameters:

_plt_ind – palette index (0-3)


Returns:

the array of color indices or null in case of incorrect palette index




Int32 spd_num_sprites()



Get the number of sprites in a project.


Returns:

the number of sprites in a project




spd_sprite_data spd_get_sprite_data( Int32 _spr_ind )



Get a sprite data by index.


Parameters:

_spr_ind – sprite index


Returns:

the spd_sprite_data object or null in case of an error




Long spd_export_CHR_data( Int32 _spr_ind, String _filename, Bool _save_padding )



Convert a sprite's CHR bank into a NES/SMS compatible format and save it to a file.


Parameters:

_spr_ind – sprite index

_filename – exported data filename

_save_padding – save padding data aligned to 1/2/4 KB or not ( ignored on SMS )


Returns:

the size of a saved file in bytes or -1 in case of an error



Sprite Data Structures




Sprite data class:



class spd_sprite_data

{

// string name

public String name;


// offset from origin

public Int32 offset_x;

public Int32 offset_y;


// sprite dimensions in pixels

public Int32 width;

public Int32 height;


// attributes array ( can be None )

public spd_sprite_attr[] attrs;


// 1x1 tiles array ( can be None )

public spd_CHR_bank CHR_bank;

}




Sprite attribute class:



class spd_sprite_attr

{

// CHR coordinates

public Int32 x;

public Int32 y;


// palette index 0-3

// SMS: ignored

public Int32 palette_ind;


// CHR index

public Int32 CHR_ind;


// horizontal flip = 0x01

// vertical flip = 0x02

// SMS: ignored

public Int32 flip_flag;

}




CHR bank class:



class spd_CHR_bank

{

// unique id

public Int32 id;


// the number of sprites that share a CHR bank

public Int32 link_cnt;


// array of 1x1 tiles

public spd_CHR_data[] CHR_data;

}




CHR data class:



class spd_CHR_data

{

// 8x8 bytes array

// NES: 0-3 color indices

// SMS: 0-15 color indices

public Byte[] data;

}
